# AI Notes Hub

Full-stack AI notes app for developers.  
**Phase 1**: React + Vanilla CSS  
**Phase 2**: FastAPI backend  
**Phase 3**: LLM features (summarize, search)

## Features
- Add, edit, delete notes  
- Responsive design (mobile-first)  
- Vanilla CSS (Grid, Flexbox, animations)

## Run locally
```bash
npm install
npm start
